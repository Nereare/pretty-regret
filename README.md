## Installing

1. Install [Yarn](https://yarnpkg.com/en/);
2. Clone the repository with `git clone https://github.com/Nereare/pretty-regret.git`;
3. Get the depedencies with `yarn` or `yarn install`;
4. Open the `index.html` file.
